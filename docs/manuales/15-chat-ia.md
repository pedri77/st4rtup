# Manual de Usuario: Chat IA

## Descripcion

El Chat IA proporciona un asistente de inteligencia artificial integrado en el CRM para consultas sobre leads, normativas, estrategias de venta y ciberseguridad GRC.

## Acceso

Hay dos formas de acceder:
- **Pagina completa**: Navegar a **Chat IA** en la barra lateral o `/chat`
- **Widget flotante**: Pulsar el boton circular en la esquina inferior derecha (disponible en cualquier pagina)

## Pagina de Chat

La pagina completa ofrece:
- Panel lateral con historial de conversaciones
- Area principal de chat
- Selector de proveedor de IA
- Gestion de conversaciones

### Conversaciones

- **Nueva conversacion**: Pulsar el boton **+** o **Nueva conversacion**
- **Historial**: Las conversaciones anteriores se listan en el panel lateral
- **Eliminar**: Pulsar el icono de papelera en una conversacion

### Proveedores

El chat soporta multiples proveedores de IA configurados en la seccion de Integraciones:
- OpenAI (GPT)
- Anthropic (Claude)
- Otros proveedores compatibles

## Widget flotante

El widget permite chatear sin salir de la pagina actual:
1. Pulsar el boton flotante (esquina inferior derecha)
2. Se abre un panel de chat compacto
3. Escribir el mensaje y pulsar Enter o el boton de enviar
4. Para cerrar, pulsar la X o el boton flotante

### Sugerencias rapidas

Al iniciar una conversacion, el widget muestra sugerencias:
- "Redacta un email de seguimiento para NIS2"
- "Argumentos de venta para DORA en banca"
- "Compara ISO 27001 vs ENS"

## Funcionalidades del chat

El asistente puede ayudar con:
- Redaccion de emails comerciales
- Argumentarios de venta por normativa
- Comparativas entre marcos regulatorios
- Estrategias de seguimiento
- Informacion sobre ciberseguridad GRC
- Analisis de leads y oportunidades

## Formato de respuestas

El chat soporta formato enriquecido:
- **Negrita** para enfasis
- `Codigo` para terminos tecnicos
- Bloques de codigo para contenido estructurado

## Consejos

- Ser especifico en las preguntas para obtener mejores respuestas
- Usar el widget para consultas rapidas sin cambiar de pagina
- Crear conversaciones separadas para temas distintos
- Configurar al menos un proveedor de IA en Integraciones
