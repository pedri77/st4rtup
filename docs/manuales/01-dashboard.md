# Manual de Usuario: Dashboard

## Descripcion

El Dashboard es la pantalla principal del CRM Riskitera Sales. Ofrece una vista consolidada de los indicadores clave de rendimiento (KPIs), graficos de actividad y resumen del estado comercial.

## Acceso

Navegar a **Dashboard** en la barra lateral o acceder a `/dashboard`.

## Indicadores Clave (KPIs)

El panel superior muestra tarjetas con metricas en tiempo real:

| KPI | Descripcion |
|-----|------------|
| Total Leads | Numero total de leads en el sistema |
| Leads Activos | Leads con estado activo en seguimiento |
| Oportunidades | Oportunidades abiertas en el pipeline |
| Valor Pipeline | Valor total en EUR de las oportunidades |
| Acciones Pendientes | Tareas pendientes de completar |
| Tasa de Conversion | Porcentaje de leads convertidos |

## Graficos

- **Actividad reciente**: Grafico de barras con actividad por dia/semana
- **Pipeline por etapa**: Distribucion de oportunidades por fase
- **Leads por estado**: Desglose visual del estado de los leads
- **Rendimiento mensual**: Evolucion de metricas clave

## Filtros

- **Rango de fechas**: Seleccionar periodo de analisis
- **Comercial**: Filtrar por usuario asignado (solo admin)

## Acciones rapidas

Desde el Dashboard se puede acceder directamente a:
- Crear nuevo lead
- Ver acciones pendientes
- Acceder al pipeline

## Consejos

- Revisar el Dashboard al inicio de cada jornada para priorizar el trabajo
- Los KPIs se actualizan automaticamente cada 60 segundos
- Usar los filtros de fecha para comparar periodos
